To download and run locally:
1. Make sure you have Node.js installed
2. Pull/Download the files into a local directory
3. In this directory (`master`) run the command `npm i` to intall the Node modules
4. In the `client` directory, run `npm i` again to intall the React modules
5. In the `server` directory, run the command `node index.js` to start the Express server
6. In the `client` directory, run the command `npm run start` to start the development React build.
7. Once both files are running concurrently, CORS-protected API calls can be made between the frontend and backend.
